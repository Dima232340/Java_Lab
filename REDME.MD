# Многопоточная система работы ресторана

Лабораторная работа по Java. Моделирование работы ресторана с официантами и поварами.

## Описание
Система моделирует работу ресторана, где:
- Клиенты генерируют заказы
- Официанты принимают заказы
- Повара готовят блюда
- Все процессы работают параллельно

## Архитектура
- **Order** - класс заказа
- **OrderQueue** - синхронизированная очередь заказов
- **Waiter** - поток официанта
- **Chef** - поток повара
- **RestaurantMonitor** - мониторинг системы
- **Restaurant** - главный класс

## Технологии
- Java 8+
- Многопоточность (Thread, ExecutorService)
- BlockingQueue для синхронизации
- ConcurrentHashMap для потокобезопасных коллекций
